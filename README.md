# SkillRoute

**SkillRoute** — это веб-платформа для развития карьеры, призванная устранить разрыв между академическими знаниями и реальными требованиями индустрии. Приложение анализирует текущий набор навыков студента, сравнивает его с требованиями конкретной вакансии и автоматически формирует персонализированную **Дорожную карту обучения** для закрытия выявленного **Пробела в навыках**.

## Основные возможности

* **Верификация через GitHub API:** Уникальная система проверки, которая анализирует репозитории пользователя на GitHub и подтверждает его технические навыки на основе реальной активности в коде.
* **Умный анализатор пробелов в навыках:** Основной движок, вычисляющий разницу между текущим уровнем компетенций студента и ожиданиями работодателя.
* **Оркестратор дорожных карт:** Автоматически создаёт пошаговый план обучения с подборкой образовательных ресурсов для каждого недостающего навыка.
* **Разграничение доступа по ролям:**
  * **Студент:** Просмотр вакансий, анализ недостающих навыков и отслеживание профессионального роста.
  * **Компания / HR:** Управление профилями вакансий, определение технического стека и установка требуемых уровней владения.
* **Контейнеризированная инфраструктура:** Полностью оркестрированная среда на базе Docker для бесперебойного развёртывания.

## Технологический стек

* **Бэкенд:** Java 17, Spring Boot 3.x
* **Безопасность:** Spring Security
* **Интеграция с API:** GitHub REST API (для верификации навыков)
* **База данных:** PostgreSQL
* **Инфраструктура:** Docker, Docker Compose
* **ORM и миграции:** Spring Data JPA, Hibernate, Liquibase
* **Слой представления:** FreeMarker (FTL)
* **Логирование:** SLF4J / Logback
* **Инструмент сборки:** Gradle

## Архитектурные особенности

* **Интеграция с внешним API:** Использует GitHub API для получения данных о репозиториях и языках программирования, обеспечивая подтверждение навыков на основе реальных данных.
* **Докеризированная среда:** Использует `docker-compose` для управления мультиконтейнерной конфигурацией (приложение + PostgreSQL), гарантируя единообразие окружения.
* **Слоистая архитектура:** Строгое разделение ответственности (Контроллеры → Сервисы → Репозитории → DTO).
* **Паттерн «Сервис-оркестратор»:** Сложная логика — например, вычисление пробела в навыках и генерация дорожной карты — вынесена в специализированные оркестраторы.

## Начало работы

### Требования

* Docker и Docker Compose
* Персональный токен доступа GitHub (для функции верификации навыков)

### Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/Arinkmm/SkillRoute.git
   cd SkillRoute
   ```

2. **Настройка окружения:**
   Создайте файл `.env` или обновите `docker-compose.yml`, указав учётные данные базы данных и токен GitHub API.

3. **Запуск через Docker Compose:**
   ```bash
   docker-compose up --build
   ```
   Приложение будет доступно по адресу `http://localhost:8080`, база данных инициализируется автоматически.
